# 二手车价格预测 - 完整优化方案总结报告

## 📋 项目概况

本项目是一个**二手车价格预测**竞赛方案，通过系统化的特征工程和模型优化，实现了显著的性能提升。

### 数据规模
- **训练集**：150,000 行，31 列
- **测试集**：50,000 行，30 列
- **目标变量**：price（二手车价格，单位：元）

---

## 🎯 优化方案总结（5大优化方向）

### **[1/5] ✅ 特征工程（高级版）**

#### 1.1 时间特征（5个）
```
✓ car_age_segment     - 车龄分段（0-3年、3-5年、5-10年、10+年）
✓ km_per_year         - 年均里程（反映车况质量）
✓ reg_season_encoded  - 注册季节编码
✓ reg_year_period     - 注册年份分组
✓ time_diff           - 注册到上线延迟（库存周期）
```

#### 1.2 品牌特征（4个）
```
✓ brand_avg_price     - 品牌平均价格（市场定位）
✓ brand_price_std     - 品牌价格标准差（价格离散度）
✓ brand_count         - 品牌列表数量（市场热度）
✓ brand_tier_encoded  - 品牌分级（豪车/经济/预算）
```

#### 1.3 车况特征（5个）
```
✓ km_segment_encoded      - 里程分段离散化
✓ power_segment_encoded   - 功率分级
✓ power_km_ratio          - 性价比特征（功率/里程）
✓ power_to_brand_price    - 功率相对品牌价格
✓ log_km, log_power       - 对数变换（处理右偏）
```

#### 1.4 异常标志特征（5个）
```
✓ notRepaired_is_missing  - 缺失值标记
✓ km_per_year_high        - 年均里程过高标记
✓ new_car_high_km         - 新车高里程异常
✓ power_is_zero           - 功率为0标记
✓ long_listing_delay      - 长期库存标记
```

#### 1.5 交互特征（4个）
```
✓ power_age       - 功率 × 车龄
✓ km_age          - 里程 × 车龄
✓ high_power_young - 高性能年轻车标记
✓ low_km_young    - 低里程年轻车标记
```

#### 1.6 分类特征编码
```
✓ bodyType_encoded      - 车体类型（目标编码）
✓ fuelType_encoded      - 燃油类型（目标编码）
✓ gearbox_encoded       - 变速箱类型（目标编码）
✓ regionCode_encoded    - 地区代码（目标编码）
```

**特征工程成果**：31 个原始特征 → 56 个精心设计的特征

---

### **[2/5] ✅ 地区特征工程（新增）**

#### 2.1 地区统计特征
```
✓ region_avg_price      - 地区平均价格
✓ region_price_std      - 地区价格标准差
✓ region_count          - 地区样本数量
✓ region_multiplier     - 地区价格乘数（相对全体均值）
```

#### 2.2 品牌-地区交叉特征
```
✓ brand_region_avg_price - 特定品牌在特定地区的均价
                          （反映地域性价差异）
```

**作用**：
- 捕捉不同地区的价格差异（北上广 vs 小城市）
- 同品牌在不同地区的定价策略差异
- 帮助模型学习地域化的价格规律

### **[3/5] ✅ 特征选择（智能筛选）**

#### 3.1 特征选择流程
1. 使用快速XGBoost模型（100轮，浅树）评估特征重要性
2. 计算特征重要性的累积百分比
3. **保留重要性最高的特征，覆盖总重要性的85%**
4. 移除低重要性特征（减少过拟合风险）

### **[4/5] ✅ 目标变量变换（Log变换）**

#### 4.1 变换原理
```
原始价格分布：
  - 均值 = 5,923.33 元
  - 中位数 = 3,250 元
  - 分布：严重右偏（有昂贵的豪车拉高均值）

Log变换后：
  - 均值 = 8.04
  - 中位数 = 8.09
  - 分布：接近正态分布
```

#### 4.2 变换公式
```python
y_train_transformed = np.log1p(y_train)  # log(1 + price)
y_pred_original = np.expm1(y_pred)      # exp(y_pred) - 1
```

**作用**：
- ✅ 处理目标变量的右偏分布
- ✅ 减少异常值（豪车）的影响
- ✅ 提高模型的预测稳定性
- ✅ 改善RMSE等对异常值敏感的指标


#### 5.2 集成权重（基于验证集R²）
```
XGBoost  权重：33.51% (R² = 0.9047)
LightGBM 权重：33.42% (R² = 0.9023)
```

#### 5.3 集成预测
```python

                   weight_lgb * y_pred_lgb +
```

**作用**：
- ✅ 充分发挥不同模型的优势
- ✅ 降低单一模型的过拟合风险
- ✅ 提高预测的稳定性和鲁棒性
- ✅ 获得更平衡的性能

## 💡 关键技术亮点

### 1️⃣ **目标编码**（Target Encoding）
```
原理：用每个分类值对应的目标变量平均值替换
优势：直接捕捉分类特征与价格的关系
实施：bodyType, fuelType, gearbox, regionCode
```

### 2️⃣ **统计特征**（Aggregation Features）
```
品牌均价、地区均价、品牌-地区交叉特征
反映市场供需关系和地域差异
```

### 3️⃣ **异常标记**（Anomaly Flags）
```
标记异常数据而非删除
让模型自主学习如何处理异常
提高模型鲁棒性
```

### 4️⃣ **对数变换**（Log Transformation）
```
处理目标变量右偏分布
减少离群点影响
改善数值稳定性
```

### 5️⃣ **模型集成**（Model Ensemble）
```
三个GBDT模型的加权投票
基于验证集R²自动计算权重
充分发挥不同模型优势
```
